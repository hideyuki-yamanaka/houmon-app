# Google Sheets で houmon-app のデータを見る

Supabase のデータを Google Sheets に **30分おきに自動コピー** する仕組み。
坂本さん・森さんとかに閲覧URLを渡して共有する用途。

## 📋 セットアップ手順（15分くらい）

### 1. 新しい Google Sheets を作る

1. <https://sheets.new> を開く（新規スプレッドシートが立ち上がる）
2. タイトルを **「家庭訪問アプリ データ」** にする

### 2. Apps Script を開いて、コードを貼る

1. メニュー **「拡張機能」 > 「Apps Script」** をクリック
2. 新しいタブで GAS エディタが開く（タイトルが「無題のプロジェクト」になってるはず）
3. エディタの初期コード（`function myFunction()...`）を **全部削除**
4. このフォルダの `sync-from-supabase.gs` を開いて中身を全コピー → GAS エディタに貼り付け
5. **フロッピーディスクアイコン** を押して保存（Cmd+S でもOK）

### 3. 最初の1回だけ手動で実行＆権限許可

1. GAS エディタ上部のプルダウンで関数 **「buildSampleProfileSheet」** を選ぶ（⭐サンプル作成用の関数）
2. **「実行」** ボタンを押す
3. **権限リクエスト**が出るので：
   - 「権限を確認」→ Googleアカウント選択
   - 「このアプリは Google で確認されていません」→ **「詳細」 → 「(安全ではないページ)に移動」**
     （自分が書いたコードなので OK）
   - 「許可」をクリック
4. 数秒待つと Sheets 側に **「【サンプル】東栄本部」** タブができる 🎉
5. 「サンプル完成！」のアラートが出たらOK

### 4. サンプルのレイアウトを確認する

1. Sheets で **「【サンプル】東栄本部」** タブを開く
2. **メニュー > ファイル > 印刷** で **A4縦** のプレビューを確認
3. 1人あたり A4 1ページに収まってるか見る
4. 見栄えOKなら → 次のステップへ
5. 「ここ変えたい」があれば ひでさんに伝える

### 5. 全本部分を作成

Sheets のメニューバーに **「houmon-app」** という項目があるはず。

1. **houmon-app > 📋 全本部のプロフィール作成** をクリック
2. 数秒〜数十秒で全本部のタブが一気に作成される
3. 本部ごとのタブが下に並ぶ（東栄本部 / 旭創価本部 / 東旭川本部 / ...）

### 6. 30分おきの自動同期をONにする

**houmon-app > ⏰ 30分おき自動同期をON** をクリック。

→ これで以降、30分ごとに **表形式の一覧シート**（メンバー/訪問ログ）が自動更新される。
   プロフィール形式のシートは手動更新（「📋 全本部のプロフィール作成」を再実行）。

### 7. 共有設定

1. 右上の **「共有」** ボタン
2. 今回の選択: **「リンクを知ってる全員」に「閲覧者」** で共有
3. ⚠️ 個人情報満載やから、URL 流出に気をつけて。本当は「特定のメールだけ」を推奨。

---

## 🔄 運用メモ

- **Sheets は読み取り専用**: 編集は houmon-app 側でやる。Sheets で書き換えても次の同期で消える。
- **プロフィール形式を更新したい時**: **houmon-app > 📋 全本部のプロフィール作成** を再実行
- **表形式を即時更新したい時**: **houmon-app > 📊 表形式で全件同期(従来版)**
- **自動同期を止めたい時**: **houmon-app > ⏹ 自動同期をOFF**
- **設定を変えたい時**: GAS エディタで `SUPABASE_URL` / `SUPABASE_ANON_KEY` を編集して保存

## 🖨 印刷時のおすすめ設定

1. **ファイル > 印刷**
2. **用紙サイズ: A4**
3. **向き: 縦**
4. **スケール: 標準（100%）** もしくは「幅に合わせる」
5. **余白: 標準**

1人あたり46行の固定レイアウトになってるから、A4 1ページにきれいに収まる設計や。
